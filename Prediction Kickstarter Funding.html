Predicting if a Kickstarter project will be funded

Oct 2019 ~ Minor Course "Introduction to Data Science"

Length:   1mo (at 0.20 FTE)

Programming language:   Python (Pandas, NumPy, Matplotlib, seaborn, scikit-learn)

Data:  370 000 Kickstarter projects, each with 15 variables, such as the category of the project, a short description, the fundraising goal, and the deadline for crowdfunding

Problem description:
Train a Machine Learning model to determine if an unseen project will receive enough funds to meet its goal

Approach & Results:
After data was explored, multiple additional attributes were derived, such as the time left to raise funds or the number of words in the name of the project. Then, the One-Hot Encoding technique was applied to the multi-categorical variables. Next, two Machine Learning classification models were trained and validated (using 10-fold Cross Validation), namely Logistic Regression (using Recursive Feature Elimination in the Feature Engineering phase) and Random Forest (using Grid Search to tune the hyperparameters). Consequently, it was found that the accuracy of the models on the test dataset is 63% for the Logistic Regression model and 74% for the Random Forest one.
