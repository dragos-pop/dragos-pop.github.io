Comparison of 3 Machine Learning models on predicting house prices

Mar 2020 ~ BSc Course "Machine Learning"

Length:   2mo (at 0.25 FTE)

Programming language:   Python (Pandas, NumPy, Math, Matplotlib, seaborn, scikit-learn, LightGBM)

Data:   3000 instances split in half for training and testing sets, each with 76 explanatory variables, describing various details of a property in Ames, Iowa, ranging from the size of the living area in square feet to the type of street that accesses the estate

Problem description:
Build, train, tune, and test three Machine Learning models, namely Linear Regression, Random Forest, and Gradient Boosting, comparing their prediction power (RMSLE) on the house prices

Approach & Results:
   - The Linear Regression model was trained using the log transformation of the top 9 features with the highest correlation with the sale price and scored an RMSLE of 0.1652 on the test data
   - The Random Forest model was built on complete data. Its hyperparameters were tunned using Random Search and validated following a 10-fold Cross-Validation. It resulted in an RMSLE of 0.1632 on the test data
   - Lastly, the Random Search technique was used again to tune the Gradient Boosting hyperparameters, validating the results with the help of 10-fold Cross-Validation. Accordingly, the RMSLE of the Gradient Boosting on the test data was 0.1675
